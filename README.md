# FaceLib 🎭

Локальний додаток для розпізнавання облич в архіві фото.

![Python](https://img.shields.io/badge/Python-3.10+-blue)
![FastAPI](https://img.shields.io/badge/FastAPI-latest-green)
![License](https://img.shields.io/badge/license-MIT-blue)

## Можливості

- 🔍 Автоматичне розпізнавання та групування облич (InsightFace/ArcFace)
- 📁 Підтримка великих архівів (перевірено на 286 000+ фото)
- 🗂 Перегляд фото по роках з навігацією по шкалі часу
- 👥 Злиття, розділення та перейменування груп людей
- ⭐ Обрані люди з drag&drop сортуванням
- 🔎 Пошук за іменем та за фото (розпізнавання обличчя)
- 🖼 Слайдшоу з зумом та перетягуванням
- 💾 Локальна база SQLite — всі дані на вашому комп'ютері
- 🇺🇦 Підтримка кирилиці в шляхах до файлів

## Вимоги

- Windows 10/11
- Python 3.10+ — [скачати з python.org](https://www.python.org/ftp/python/3.11.9/python-3.11.9-amd64.exe) *(якщо не встановлений — install.bat встановить автоматично)*
- 4+ GB RAM (8+ GB рекомендовано для великих архівів)
- NVIDIA GPU (опційно, для прискорення)

## Встановлення

1. Натисніть зелену кнопку **Code → Download ZIP**
2. Розпакуйте архів у будь-яку папку
3. Запустіть `install.bat` — встановить всі залежності і створить ярлик **FaceLib** на робочому столі
4. Двічі клікніть на ярлик **FaceLib** — програма відкриється в браузері автоматично

> **Примітка:** Якщо Python не встановлений — install.bat скачає і встановить його автоматично. Потрібне інтернет-з'єднання.

## Швидкий старт

1. Додайте папку з фото через інтерфейс
2. Натисніть **Індексувати** — програма знайде всі обличчя
3. Обличчя автоматично згрупуються по людях
4. Підпишіть людей іменами і насолоджуйтесь пошуком!

## Стек

- **Backend:** Python, FastAPI, InsightFace, SQLite
- **Frontend:** Vanilla HTML/JS (без фреймворків)
- **Розпізнавання:** ArcFace (InsightFace)

## Структура проекту

```
facelib/
├── app.py              # FastAPI сервер
├── index.html          # Веб-інтерфейс
├── install.bat         # Встановлення + ярлик на робочому столі
├── start.bat           # Запуск з логами (для діагностики)
├── start.vbs           # Запуск у фоні (використовується ярликом)
├── facelib.ico         # Іконка програми
└── facelib.db          # База даних (створюється автоматично)
```
